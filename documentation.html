FLIX

Intro

Objective To build the server­side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies. Context It’s no longer enough for a JavaScript developer to be skilled in frontend development alone. It’s become essential that you be able to interface with (as you did in Achievement 1) and even create your own APIs. For this reason, throughout this Achievement, you’ll create a REST API for an application called “myFlix” that interacts with a database that stores data about different movies.

REQUEST URL HTTP METHOD REQUEST BODY DATA FORMAT RESPONSE BODY DATA FORMAT
Get a list of all movies /movies GET None JSON object holding data about all movies.
Get a single movie by title /movies/:title

Example:

/movies/Iron Man

 GET None JSON object holding data about a single movie containing a title, description, genre, director and image URL.

The genre of Iron man is Sci-fi

Get data about a director by name /directors/:name GET None JSON object holding data about a director, including his bio, birth year, death year.
Allow new users to register /users POST A JSON object holding data about the user to add, structured like:

{
username: 'enyelsequeira',
email: 'user@mail.com',
password: '1234movie',
dateOfBirth: '04/04/1994',

A JSON object holding data about the user that was added.
Update user log in information /users/:username/:password PUT None A text message indicating whether the user's username/password was successfully updated
Update user personal information /users/:username/:email/:dateofbirth PUT None A text message indicating whether the user's personal data was successfully updated
Add a movie to the user's list of favorites /users/:username/:favorites POST A JSON object holding data about the user to add, structured like:

A JSON object holding data about the user that was added, including a unique ID:

{

Remove a movie from user's favorites /users/:username/:favorites DELETE None A text message indicating whether the movie was successfully removed from user's favorites.
Deregister existing user /users/:username DELETE None A text message indicating whether the user was successfully deregistered.
